# 🎉 Birthday Surprise Website

A personalized birthday surprise website created with HTML, featuring a beautiful slideshow, background music, and heartfelt messages.

## ✨ Features

* 📸 Image slideshow
* 🎵 Background birthday music
* 💖 Love message animation
* 🌐 Fully responsive design

## 🚀 Live Demo

👉 https://happy-birthday-urvashi.netlify.app/

## 📁 Project Structure

* `index.html` – Main webpage
* `*.jpg` – Images for slideshow
* `birthday-song-for-urvashi.mp3` – Background music

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
